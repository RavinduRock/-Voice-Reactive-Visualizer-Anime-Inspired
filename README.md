# 🎧 Voice Reactive Visualizer – Gojo Inspired

An interactive audio-based visual system built in Processing that reacts to voice intensity and duration to trigger different visual states inspired by Jujutsu Kaisen.

---

## ⚡ Features

* 🎤 Real-time microphone input detection
* 📊 Volume-based interaction system
* 🎨 Dynamic visual transitions based on sound duration
* 🌀 Multiple visual modes inspired by Gojo's abilities:

  * **Blue** – Short input bursts
  * **Red** – Medium intensity input
  * **Hollow Purple** – Sustained high input
  * **Infinite Void** – Extreme duration/peak input state

---

## 🧠 System Logic

The system analyzes:

* Input **volume**
* Duration of sustained sound
* Peak thresholds

These values determine which visual state is triggered, creating a layered interaction system instead of simple sound reactivity.

---

## 🛠 Built With

* Processing (Java)
* Sound & Video Libraries

---

## 🎮 Controls

* Speak/make sound to trigger visuals
* Different durations unlock different abilities

---

## 🚀 Purpose

This project explores how **audio input can drive gameplay-like systems**, simulating ability activation through player intent (voice control).
